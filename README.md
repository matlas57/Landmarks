# Landmarks
## SwiftUI Tutorial Project
This project is a demo app for iOS from the [SwiftUI Tutorials](https://developer.apple.com/tutorials/swiftui) by Apple.
It uses the following features:
* SwiftUI elements
* MapKit
* CoreLocation Coordinates
* JSON Decoders
* Navigation
* State
* Environment property wrapper and modifier
* Bindings
* CoreGraphics
* Animations and transitions
* UIKit
